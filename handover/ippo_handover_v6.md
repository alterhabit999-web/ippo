## 1. プロジェクト概要

毎日朝・夜2回、1〜2分で記録する自己振り返りアプリ。気分・体調・睡眠・感情のパターンを蓄積・可視化し、自己理解を深めることが目的。まず個人用Webアプリとして作り、最終的にApp Store / Google Playへのローンチを目指す。

## 3. 確定済み事項

### アプリ名
**iPPO**（"i" は小文字）

### コンセプト
「一歩一歩、自分と向き合う」。シンプルだが手書きのような温もり。開いたときに「ほっとする＋ちょっと楽しい」気持ちになれるトーン。

### 記録項目

**朝の記録（5項目）**
1. 睡眠時間 — 数値入力（スマホ数字キーボード）
2. 睡眠の質 — 5段階スライダー
3. 今朝の気分 — 5段階スライダー
4. 今朝の体調 — 5段階スライダー
5. 今日の自分との約束 — テキスト入力

**夜の記録（6項目）**
1. 今日の気分・感情 — 5段階スライダー（必須）
2. エネルギー・生産性 — 5段階スライダー（必須）
3. ストレスレベル — 5段階スライダー（必須）
4. 今日のHAPPY — テキスト入力（任意）
5. 今日のモヤモヤ — テキスト入力（任意）
6. 明日の自分との約束 — テキスト入力（必須）

### タイポグラフィ
- アプリ名・大見出し：Georgia（セリフ体）、letter-spacing 広め
- 本文・入力：サンセリフ、weight 400
- ラベル：サンセリフ、weight 500、letter-spacing 広め
- フォントカラーは真っ黒を使わない

### 画面構成

**メイン画面（3ステート）**
- ステート1：中央にボタン一つ（"Morning" / "Night"）＋「タップして記録をはじめる」＋朝夜の完了状態表示
- ステート2：記録画面（スライダー＋テキスト入力）
- ステート3：待機画面（今日の一言＋朝夜の完了状態表示）
- 17時を境に朝・夜を自動判定
- 記録済みの場合は自動で待機画面へ

**サイドメニュー**：カレンダー・インサイト・設定

**カレンダー画面**
- 月グリッド、ドット（オレンジ=朝/水色=夜）で記録状況を表示
- タップで詳細パネル（Morning/Nightタブ切替・AMテーマ/PMテーマで統一）

**インサイト画面**
- 期間切替：7日 / 30日 / 全期間（横スクロール対応）
- グラフ①：気分・エネルギー・ストレス折れ線（凡例タップで絞り込み）
- グラフ②：睡眠時間の推移（3h刻み縦軸）
- グラフ③：睡眠時間との相関性（気分・エネルギー・ストレスの3軸散布図）
- 上部に期間平均サマリーカード
- データがない場合は「記録を始めるとグラフが表示されます」と表示

**設定画面**
- 通知：朝・夜のトグル＋時刻変更
- 今日の一言：3件まで表示→「一覧を表示」で全件展開・追加・編集・削除・出典入力
- データ：CSVエクスポートボタン（**実装済み**）

### データ保存
- **保存先**：iPhoneのSafariのlocalStorage
- **キー形式**：`ippo:records:{日付}:{am or pm}`
- **注意**：Safariの「履歴とWebサイトデータを消去」でデータが消える

### CSVエクスポート仕様
- 設定画面「データ」セクションのボタンをタップでダウンロード
- ファイル名：`ippo_records_YYYY-MM-DD.csv`（実行日の日付）
- BOM付きUTF-8（ExcelでもWindows/Macで文字化けしない）
- 列構成：日付 / 朝5項目 / 夜6項目（1日1行）
- テキスト内のカンマ・改行・ダブルクォートは自動エスケープ
- タップ後3秒間「エクスポートしました ✓」とフィードバック表示
- 記録ゼロの場合はアラートを表示
- **iPhoneのSafariでは**ダウンロードフォルダ保存 or 新規タブ表示になる場合あり。PCブラウザでの動作が安定。

## 4. 環境構築済みの設定

**開発環境**
- Mac + VSCode + Node.js
- プロジェクト場所：デスクトップの `ippo` フォルダ
- 起動コマンド：ターミナルで `cd ~/Desktop/ippo` → `npm start`
- iPhone確認：同一Wi-Fiで `http://[MacのIPアドレス]:3000`
- MacのIPアドレス確認：`ipconfig getifaddr en0`

**iPhone表示対応（適用済み）**

`public/index.html`:
```html
<meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1, user-scalable=no" />
```

`src/App.css` 先頭:
```css
* { box-sizing: border-box; margin: 0; padding: 0; }
html, body { width: 100%; max-width: 100vw; overflow-x: hidden; }
```

`src/App.js` の主な設定:
- `Phone` 関数：`width:100%, minHeight:100vh` の全画面表示
- インサイト・設定・カレンダー画面：`flex:1` でフル高さ対応
- カレンダー詳細パネル：`maxHeight:"75vh"`
