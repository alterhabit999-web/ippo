# 振り返りアプリ 仕様書
**バージョン**: v0.2  
**作成日**: 2026-04-08  
**ステータス**: ドラフト

---

## 1. プロダクト概要

### アプリ名
**iPPO**

### コンセプト
毎日2回・1〜2分で記録し、自分の気分・体調・感情のパターンを可視化するパーソナル振り返りアプリ。記録の負荷を最小限に抑えながら、蓄積データから自己理解を深めることを目的とする。

### ターゲットユーザー
- まず自分自身（個人利用）
- 将来的：自己理解・セルフケアに関心を持つ一般ユーザー

### 目標
- フェーズ1：個人用プロトタイプの完成
- フェーズ2：アプリストア（App Store / Google Play）へのローンチ

## 2. 記録項目定義（確定版）

### 朝の記録（起床後すぐ・目安60秒）

| # | 項目名 | 入力形式 | 説明 |
|---|--------|----------|------|
| 1 | 睡眠時間 | 数値入力（時間） | 昨夜の睡眠時間（例：6.5） |
| 2 | 睡眠の質 | 5段階スライダー | 1=最悪 〜 5=最高 |
| 3 | 今朝の気分 | 5段階スライダー | 1=最悪 〜 5=最高 |
| 4 | 今朝の体調 | 5段階スライダー | 1=最悪 〜 5=最高 |
| 5 | 今日の自分との約束 | テキスト入力（一言） | 今日大切にしたい一つのこと |

### 夜の記録（就寝前・目安60〜90秒）

| # | 項目名 | 入力形式 | 必須 | 説明 |
|---|--------|----------|------|------|
| 1 | 今日の気分・感情 | 5段階スライダー | 必須 | 1=最悪 〜 5=最高 |
| 2 | エネルギー・生産性 | 5段階スライダー | 必須 | 1=低い 〜 5=高い |
| 3 | ストレスレベル | 5段階スライダー | 必須 | 1=なし 〜 5=強い |
| 4 | 今日のHAPPY | テキスト入力 | 任意 | 今日あったポジティブなこと |
| 5 | 今日のモヤモヤ | テキスト入力 | 任意 | 今日あったネガティブなこと |
| 6 | 明日の自分との約束 | テキスト入力（一言） | 必須 | 明日の自分へのメッセージ |

## 3. 画面構成

### 3-1. 画面一覧

| 画面名 | 種別 | 役割 |
|--------|------|------|
| メイン画面 | 常時表示 | 記録状態によって3ステートを切り替え |
| 朝の記録画面 | メインから遷移 | 朝5項目の入力フォーム |
| 夜の記録画面 | メインから遷移 | 夜6項目の入力フォーム |
| カレンダー画面 | サイドメニューから | 日別の記録状況・過去の記録閲覧 |
| インサイト画面 | サイドメニューから | グラフ・パターン分析・傾向表示 |
| 設定画面 | サイドメニューから | 通知時刻・プロフィール設定 |

### 3-2. メイン画面の3ステート

アプリ起動時に表示される画面は、記録状況と時間帯によって自動で切り替わる。

#### ステート1：記録待ち（未記録）
- 画面中央にボタン一つのみ表示。ボタン内テキストは "Morning"（朝）/ "Night"（夜）のみ
- 時間帯によって朝・夜を自動判定（判定時刻は設定で変更可）
- 朝モード：ウォームオレンジベース × 白カード配色
- 夜モード：深夜ブルー × ラベンダー配色

#### ステート2：記録画面
- ステート1のボタンを押すと遷移
- 項目を縦に並べたシンプルなフォーム
- スライダー（5段階）＋テキスト入力の構成
- 「記録する」ボタンで保存・ステート3へ

#### ステート3：待機画面（記録済み）
- 今日の記録が完了している状態
- 画面中央に「今日の一言」（ポジティブな言葉）をランダム表示
- 朝・夜両方済んだ場合も同じ待機画面
- 左上のハンバーガーメニュー（☰）でサイドメニューを開く

### 3-3. サイドメニュー
- 画面左からスライドインして表示
- メニュー項目：カレンダー、インサイト、設定
- 背景をタップで閉じる

### 3-4. サブ画面の詳細

#### カレンダー画面
- 月カレンダー（グリッド形式）、左右矢印で月移動
- 各日付の下にドット表示：オレンジ（朝の記録あり）・水色（夜の記録あり）
- 朝・夜両方記録済みの日は日付背景をサブカラーで塗りつぶし
- 今日の日付はアクセントカラーで枠線表示
- 記録済みの日をタップすると詳細パネルが下からスライドイン
- 詳細パネルはMorning / Nightのタブ形式で切り替え
  - Morningタブ：AMテーマカラー（背景・文字・カード）で統一
  - Nightタブ：PMテーマカラー（背景・文字・カード）で統一
- カレンダー自体の背景は時間帯（17時境界）でAM/PMテーマが自動切替
- 表示期間：7日 / 30日 / 90日 で切り替え
- グラフ1：気分・エネルギー・ストレスの折れ線グラフ（3軸）
- グラフ2：睡眠時間 × 翌朝の気分の散布図
- グラフ3：睡眠の質 × 日中の生産性の相関
- テキスト分析（将来機能）：HAPPY/モヤモヤのキーワード傾向

#### 設定画面
- 背景は時間帯（17時境界）でAM/PMテーマが自動切替
- 通知セクション：朝・夜それぞれトグルのオン/オフ＋時刻変更
- 「今日の一言」セクション：
  - デフォルト3件まで表示、4件以上は「一覧を表示（全○件）」ボタンで展開
  - 各言葉を編集・削除可能
  - 出典（人名・書名など）を任意で入力可能
  - 「+ 言葉を追加する」で新規追加
- データセクション：CSVエクスポートボタン

## 4. 通知仕様

| 通知 | タイミング | デフォルト時刻 | 内容 |
|------|-----------|--------------|------|
| 朝のリマインダー | 毎日朝 | 7:00 AM | 「おはようございます。今日の朝の記録をしましょう」 |
| 夜のリマインダー | 毎日夜 | 10:00 PM | 「今日もお疲れさまでした。夜の記録をしましょう」 |

- 通知時刻はユーザーが設定画面から変更可能
- 記録済みの場合は通知を出さない（UX配慮）

## 5. データ設計

### 保存するデータ（1レコード = 1回の記録）

```
記録ID, ユーザーID, 日付, 記録タイプ（朝/夜）,
睡眠時間, 睡眠の質, 今朝の気分, 今朝の体調,
今日の自分との約束,
今日の気分, エネルギー, ストレス,
今日のHAPPY, 今日のモヤモヤ, 明日の自分との約束,
作成日時
```

### 分析に使う主な相関
- 睡眠時間・質 → 翌朝の気分・体調
- 朝の気分・体調 → 夜のエネルギー・ストレス
- ストレスレベル → 気分スコアとの相関
- 朝の約束 → 夜の気分への影響（主観的追跡）

## 6. 技術方針（フェーズ別）

### フェーズ1：個人用プロトタイプ
- **プラットフォーム**：Webアプリ（スマホブラウザで動作）
- **技術**：React + ローカルストレージ or Supabase（クラウド保存）
- **目的**：実際に使いながら使い勝手を検証する

### フェーズ2：ネイティブアプリ化
- **プラットフォーム**：iOS / Android
- **技術候補**：React Native（コードを最大限再利用）
- **追加機能**：プッシュ通知、オフライン対応、App Store申請

### フェーズ3：ローンチ・拡張
- ユーザー認証（ログイン機能）
- クラウド同期・バックアップ
- AIによる週次インサイト生成（Claude API活用）
- プレミアム機能の検討

## 7. デザインシステム（確定）

### コンセプト
「一歩一歩、自分と向き合う」。シンプルだが手書きのような温もりがある。開いたときに「ほっとする＋ちょっと楽しい」気持ちになれるトーン。

### カラーパレット

#### 朝モード（案C · サンライズオレンジ）
| 役割 | カラー |
|------|--------|
| ベース背景 | `#FFFFFF`（白） |
| サブ背景 | `#FFF0E0` |
| アクセント（メイン） | `#F0A060` |
| アクセント（強調） | `#E07030` |
| テキスト | `#2C2016` |

#### 夜モード（確定・案A インクブルー）
| 役割 | カラー |
|------|--------|
| ベース背景 | `#1A2028`（インクブルー） |
| サブ背景 | `#232830` |
| ボーダー | `#343C48` |
| アクセント（メイン） | `#5A8AAA` |
| アクセント（明） | `#A8C8D8`（スカイ） |
| テキスト | `#D8E8EE` |
| テキスト（ミュート） | `#6A7888` |

### タイポグラフィ
- アプリ名・大見出し：セリフ体（Georgia）、letter-spacing 広め
- 本文・入力：サンセリフ、weight 400
- ラベル：サンセリフ、weight 500、letter-spacing 広め
