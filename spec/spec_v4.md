## 1. プロダクト概要

### アプリ名
**iPPO**

### コンセプト
毎日2回・1〜2分で記録し、自分の気分・体調・感情のパターンを可視化するパーソナル振り返りアプリ。記録の負荷を最小限に抑えながら、蓄積データから自己理解を深めることを目的とする。

### ターゲットユーザー
- まず自分自身（個人利用）
- 将来的：自己理解・セルフケアに関心を持つ一般ユーザー

### 目標
- フェーズ1：個人用プロトタイプの完成 ← **現在ここ**
- フェーズ2：アプリストア（App Store / Google Play）へのローンチ

## 2. 記録項目定義（確定版）

### 朝の記録（起床後すぐ・目安60秒）

| # | 項目名 | 入力形式 | 説明 |
|---|--------|----------|------|
| 1 | 睡眠時間 | 数値入力（時間） | 昨夜の睡眠時間（例：6.5） |
| 2 | 睡眠の質 | 5段階スライダー | 1=最悪 〜 5=最高 |
| 3 | 今朝の気分 | 5段階スライダー | 1=最悪 〜 5=最高 |
| 4 | 今朝の体調 | 5段階スライダー | 1=最悪 〜 5=最高 |
| 5 | 今日の自分との約束 | テキスト入力（一言） | 今日大切にしたい一つのこと |

### 夜の記録（就寝前・目安60〜90秒）

| # | 項目名 | 入力形式 | 必須 | 説明 |
|---|--------|----------|------|------|
| 1 | 今日の気分・感情 | 5段階スライダー | 必須 | 1=最悪 〜 5=最高 |
| 2 | エネルギー・生産性 | 5段階スライダー | 必須 | 1=低い 〜 5=高い |
| 3 | ストレスレベル | 5段階スライダー | 必須 | 1=なし 〜 5=強い |
| 4 | 今日のHAPPY | テキスト入力 | 任意 | 今日あったポジティブなこと |
| 5 | 今日のモヤモヤ | テキスト入力 | 任意 | 今日あったネガティブなこと |
| 6 | 明日の自分との約束 | テキスト入力（一言） | 必須 | 明日の自分へのメッセージ |

## 3. 画面構成

### 3-1. 画面一覧

| 画面名 | 種別 | 役割 |
|--------|------|------|
| ログイン画面 | 起動時 | Supabase Auth でメール認証 |
| メイン画面 | 常時表示 | 記録状態によって3ステートを切り替え |
| 記録画面 | メインから遷移 | 朝/夜の入力フォーム |
| 待機画面 | メインのステート3 | 記録済み時に表示。今日の一言を表示 |
| カレンダー画面 | サイドメニューから | 日別の記録状況・過去の記録閲覧 |
| インサイト画面 | サイドメニューから | グラフ・パターン分析・傾向表示 |
| 集めた言葉画面 | サイドメニューから | 登録した言葉の一覧・管理 |
| 設定画面 | サイドメニューから | 通知時刻・ログアウト |

### 3-2. メイン画面の3ステート

#### ステート1：記録待ち（未記録）
- 画面中央にボタン一つのみ表示（"Morning" / "Night"）
- 17時境界で朝・夜を自動判定
- 背景：波アニメーション

#### ステート2：記録画面
- スライダー（5段階）＋テキスト入力
- 「記録する」ボタンで保存・待機画面へ
- 保存エラー時は赤色エラーメッセージを表示

#### ステート3：待機画面（記録済み）
- 今日の一言をランダム表示、言葉の下に出典を表示
- 「別の言葉を見る」でランダム更新
- 「言葉をあつめる」ボタンで集めた言葉画面へ
- 朝夜の完了状態をドットで表示
- 背景：波アニメーション

### 3-3. 背景の仕様

#### 波アニメーション適用画面
メイン・待機・記録・カレンダー・設定画面

#### 単色背景（T.bgPage）画面
**インサイト・集めた言葉画面**
- 画面全体（ヘッダー含む）を `background: T.bgPage` のdivでラップ
- TopBarとコンテンツエリアが同じ単色で統一される
- アニメーションは背面で動いているが、上記divで完全に隠れる

#### WaveBackground コンポーネント仕様
- `position: fixed; top/right/bottom/left: -100px` で配置
  - iOSセーフエリア（ノッチ・ホームインジケーター部分）を含む全領域をカバー
  - `-100px` のオフセットで viewport 外側まで確実に塗りつぶし
- 横波レイヤー：画面幅の3倍の横グラデーション。14秒周期で左右移動（`ippoWaveH`）
- 縦波レイヤー：画面高さの3倍の縦グラデーション。22秒周期で上下移動（`ippoWaveV`）。opacity: 0.55で重ねる
- アニメーションは `transform: translateX/Y` のみ使用（GPU最適化・レイアウト再計算なし）

**朝モード波グラデーション**
- 横波：`#FFF4E8 → #FFE8CC → #F0C090 → #FFE8CC → #FFF4E8`
- 縦波：`#FFF4E8 → #FDEEE0 → #FFF4E8 → #FFF0E2`

**夜モード波グラデーション**
- 横波：`#1A2028 → #222C3C → #2A3C52 → #222C3C → #1A2028`
- 縦波：`#1A2028 → #1D2936 → #1A2028 → #1E2A38`

### 3-4. サイドメニュー
- 画面左からスライドインして表示
- メニュー項目：カレンダー、インサイト、集めた言葉、設定
- 背景をタップで閉じる

### 3-5. 各画面の詳細

#### カレンダー画面（波アニメーション）
- 月カレンダー（グリッド形式）、左右矢印で月移動
- ドット表示：オレンジ（朝の記録あり）・水色（夜の記録あり）
- 記録済みの日をタップすると詳細パネルが下からスライドイン
- 詳細パネルはMorning / Nightのタブ形式

#### インサイト画面（単色背景）
- 期間切替：7日 / 30日 / 全期間
- グラフ①：気分・エネルギー・ストレス折れ線
- グラフ②：睡眠時間の推移
- グラフ③：睡眠時間との相関性（散布図）
- 上部に期間平均サマリーカード

#### 集めた言葉画面（単色背景）
- 「今日の言葉」フィーチャー表示（出典付き）
- 「みんなの言葉」：他ユーザーの言葉をランダム表示・自分のコレクションに追加可能
- 言葉一覧（10件ごとページネーション）
- 各言葉を編集・削除可能
- 「+ 言葉を追加する」で新規追加（テキスト・出典入力）

#### 設定画面（波アニメーション）
- 通知：朝・夜のトグル＋時刻変更
- ログアウトボタン

## 5. データ設計

### 保存先（確定）
- **Supabase**（PostgreSQL）
- 認証：Supabase Auth（メール/パスワード）
- 複数ユーザー対応・RLS設定済み

### テーブル構成

**`records` テーブル**
```
id (uuid), user_id, date, type (am/pm),
sleep_hours, sleep_quality, mood_morning, condition_morning, promise_morning,
mood_evening, energy, stress, happy, worries, promise_evening,
created_at
```
UNIQUE制約: `records_user_date_type_unique`（user_id, date, type）

**`quotes` テーブル**
```
id (uuid), user_id, text, source, created_at
```
デフォルト言葉は `source = "iPPO"` で識別。初回ログイン時に自動シード。
